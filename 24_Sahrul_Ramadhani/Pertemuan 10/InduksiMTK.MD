### Penyelesaian dengan Induksi Matematika

#### Pernyataan 1
**Pernyataan:**
2 + 4 + 6 + ... + 2n = n(n + 1)

- **Basis:**
  Ketika n = 1:  
  2 = 1(1 + 1) → Benar.

- **Hipotesis:**
  Misalkan pernyataan benar untuk n = k:  
  2 + 4 + ... + 2k = k(k + 1).

- **Induksi:**
  Untuk n = k + 1:  
  2 + 4 + ... + 2k + 2(k + 1)  
  = k(k + 1) + 2(k + 1)  
  = (k + 1)(k + 2).  
  Jadi, pernyataan benar untuk n = k + 1.

Pernyataan terbukti.

#### Pernyataan 2
**Pernyataan:**
1 + 4 + 9 + ... + n² = (1/6) * n(n + 1)(2n + 1)

- **Basis:**
  Ketika n = 1:  
  1 = (1/6) * 1(1 + 1)(2(1) + 1)  
  = 1 → Benar.

- **Hipotesis:**
  Misalkan pernyataan benar untuk n = k:  
  1 + 4 + 9 + ... + k² = (1/6) * k(k + 1)(2k + 1).

- **Induksi:**
  Untuk n = k + 1:  
  1 + 4 + 9 + ... + k² + (k + 1)²  
  = (1/6) * k(k + 1)(2k + 1) + (k + 1)²  
  = (1/6) * (k + 1)(k + 2)(2k + 1).  
  Jadi, pernyataan benar untuk n = k + 1.

Pernyataan terbukti.

#### Pernyataan 3
**Pernyataan:**
2 + 6 + 12 + ... + n(n + 1) = (1/3) * n(n + 1)(n + 2)

- **Basis:**
  Ketika n = 1:  
  2 = (1/3) * 1(1 + 1)(1 + 2)  
  = 2 → Benar.

- **Hipotesis:**
  Misalkan pernyataan benar untuk n = k:  
  2 + 6 + 12 + ... + k(k + 1) = (1/3) * k(k + 1)(k + 2).

- **Induksi:**
  Untuk n = k + 1:  
  2 + 6 + 12 + ... + k(k + 1) + (k + 1)(k + 2)  
  = (1/3) * k(k + 1)(k + 2) + (k + 1)(k + 2)  
  = (1/3) * (k + 1)(k + 2)(k + 3).  
  Jadi, pernyataan benar untuk n = k + 1.

Pernyataan terbukti.
